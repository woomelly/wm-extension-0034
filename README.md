# Woomelly - Extension 0034

Woomelly extension to show Mercado Libre price in the WooCommerce products list

## Installing

1. Unpack the entire contents of this plugin zip file into your 'wp-content/plugins/' folder locally
2. Upload to your site
3. Navigate to 'wp-admin/plugins.php' on your site (your WP Admin plugin page)
4. Activate this plugin

### Prerequisites

1. Must be active the [plugin WooCommerce](https://woocommerce.com).
2. Must be active the [plugin Woomelly](https://woomelly.com).
